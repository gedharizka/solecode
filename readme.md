## How To Run

1. Clone repository
```bash
  git clone https://github.com/gedharizka/terraform.git
```

2. Install depedencies
```bash
  cd [FOLDER_NAME]
```
```bash
  npm i
```

3. Migrate database

```bash
  node database/migration.js
```

4. Setup config, fill with your env config

```bash
  cp .env.dev .env
```


5. Run Project
```bash
  npm run start
```
